Pylint设置提示显示的种类

解决使用django时代码正确却出现error的情况

首先在vscode中打开终端输入pip install django-pylint
下载完毕后按Ctrl+Shift+P然后输入setting找到*Open Setting(JSON)*选项

然后在末尾加入（记得给上一行结尾加上逗号）

之后可能会显示
Django was not configured. For more information run pylint --load-plugins=pylint_django --help-msg=django-not-configured

你可以在setting.json中的--disable一行中加上django-not-configured更改后为
"--disable=C, django-not-configured"
另外，你也可以设置"--django-settings-module=path.settings"将path改为主程序路径，然后设置settings.py
